# Indirect-b2b-marketplace
Project ini bertujuan untuk membuat fitur yang memungkinkan user untuk membuat Purchase Order (PO) secara otomatis di Odoo. Fitur ini menggunakan n8n sebagai tool untuk workflow otomasi dan komunikasi dilakukan menggunakan API. Code pada repository ini sudah saya kurasi untuk menampilkan yang saya ubah saja.
